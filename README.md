# MoneyMoney

React + TypeScript + [Vite](https://vite.dev/) + [Tailwind CSS v4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/).

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Stack

- React 19
- Vite 8
- Tailwind CSS v4 with `@tailwindcss/vite`
- shadcn/ui (Radix Nova preset, Lucide icons, Geist Variable font)
