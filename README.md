# MoneyMoney

React + TypeScript + [Vite](https://vite.dev/) + [Tailwind CSS v4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/).

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Stack

- React 19
- Vite 8
- Tailwind CSS v4 with `@tailwindcss/vite`
- shadcn/ui (Radix Nova preset, Lucide icons, Geist Variable font)

## Push to GitHub

Create an empty repo named `MoneyMoney` under your account, then from this folder:

```bash
git remote add origin https://github.com/YOUR_USERNAME/MoneyMoney.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username or organization.
